Urgent: Update required for Electrum

It became apparent this weekend that the commonly used Electrum wallet has a vulnerability that potentially allows random websites to steal the wallet file through Javascript. It’s important for users to avoid opening the Electrum wallet. If you happen to have the wallet active, it’s stringent to immediately close it. An update has been released for Electrum, which addresses the security issue.
